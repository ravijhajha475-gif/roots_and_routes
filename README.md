## 📖 About the Project

**Roots & Routes** is an e-commerce platform that connects travelers with local artisans across every Indian state. It combines tourism, logistics, and digital payments into a single seamless marketplace.

Every purchase on the platform:
- ✅ Directly supports the artisan who made the product
- ✅ Preserves centuries-old Indian craft traditions
- ✅ Delivers authentic, GI-tagged souvenirs to buyers worldwide

---

## ✨ Features

- 🗺️ **Shop by State** — Browse crafts from all 28 Indian states
- 🧑‍🎨 **Artisan Profiles** — Stories, earnings, and ratings of verified craftspeople
- 🔍 **Smart Search** — Filter by state, craft type, and price range
- 💳 **Digital Payments** — UPI, Cards, Net Banking, EMI via Razorpay
- 🚚 **Pan-India & International Delivery** — Delhivery, DTDC, India Post, DHL
- 📦 **Eco Packaging** — 100% biodegradable materials
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop

## 🚀 Getting Started

No installation needed. This is a pure HTML/CSS website.

**To run locally:**

1. Clone the repository
```bash
git clone https://github.com/YOUR-USERNAME/roots-and-routes.git
```

2. Open `index.html` in your browser
```bash
cd roots-and-routes
open index.html
```

That's it — no build tools, no dependencies!

## 🌍 Deployment

This site is hosted for free on **GitHub Pages**.

To deploy your own copy:
1. Fork this repository
2. Go to **Settings → Pages**
3. Set source to **main branch**
4. Your site will be live at `https://YOUR-USERNAME.github.io/roots-and-routes`
